# 📱 CineVerse: Videoconferência Inteligente

> Uma solução ágil e poderosa que une a inteligência do **Manus AI** com a estabilidade do **Jitsi Meet** para criar salas de reuniões instantâneas de forma simplificada.

---

## ✨ Proposta de Valor
Diferente de plataformas pesadas e burocráticas, este app foca na **velocidade e simplicidade**. 
Ele permite que usuários criem ou entrem em salas de videoconferência em segundos, utilizando o Manus AI para otimizar o fluxo de interação e o Jitsi como motor de comunicação criptografada, 
garantindo privacidade e alta performance sem a necessidade de cadastros complexos.

## 🚀 Instruções de Uso

1. **Acesse o App**: Abra o aplicativo em seu dispositivo móvel ou simulador.
2. **Crie uma Sala**: No campo de texto principal, digite o nome que deseja dar à sua reunião (ex: `Reuniao-TI-2024`).
3. **Inicie a Chamada**: Clique no botão **"Entrar na Reunião"**. O app abrirá automaticamente a interface do Jitsi Meet.
4. **Convide Participantes**: Compartilhe o nome da sala com outras pessoas para que elas entrem na mesma conferência.

## 🛠️ Tecnologias Utilizadas
* **Manus AI**: Auxílio na estruturação lógica e inteligência do fluxo.
* **React Native / Expo**: Framework para desenvolvimento mobile nativo.
* **Jitsi Meet SDK**: Motor de videoconferência open-source.
* **JavaScript**: Lógica de desenvolvimento.
