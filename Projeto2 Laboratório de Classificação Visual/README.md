# 🎮 Batalha de Modelos & Engenharia de Prompt (XML)

## 📝 Descrição do Projeto
[cite_start]Este projeto consiste em um estudo comparativo de **Engenharia de Prompt** utilizando uma estrutura baseada em **XML** para a criação de uma interface web[cite: 2, 5]. [cite_start]O objetivo principal é testar a eficácia de diferentes modelos de IA na geração de uma página HTML Single Page para uma **Loja de Jogos de Videogame e Computador**, seguindo diretrizes rígidas de design e funcionalidades[cite: 5, 7].

[cite_start]Desenvolvido como parte da disciplina de **Engenharia de Prompt e Aplicações em IA**, o experimento avalia a precisão técnica e a eficiência de ferramentas como ChatGPT, Gemini, Claude, entre outras[cite: 1, 6]. [cite_start]O sistema utiliza um prompt estruturado para garantir que a IA compreenda elementos como layout responsivo, paleta de cores específica (Preta, Azul e Branco) e seções obrigatórias como portfólio e avaliação de jogos[cite: 7].

## 🚀 Tecnologias Utilizadas
* [cite_start]**Linguagens de Saída:** HTML5 e CSS3 (Internal)[cite: 7].
* [cite_start]**Estruturação:** XML (eXtensible Markup Language) para o prompt[cite: 7].
* [cite_start]**Ferramentas de IA:** ChatGPT, Gemini, Claude, Qwen, DeepSeek, Grok e Maritaca[cite: 6, 9].
* [cite_start]**Participantes:** Rafael Silva Covre Bragante, José Rodrigues Pereira Junior e Ana Beatris Alves da Silva[cite: 3].

## 📊 Resultados e Aprendizados
[cite_start]O projeto alcançou resultados detalhados sobre a performance de cada modelo frente ao prompt estruturado[cite: 8].
* [cite_start]**Compreensão do Modelo:** O Gemini e o Grok demonstraram a maior "compreensão" da estrutura do prompt em XML[cite: 13, 14].
* [cite_start]**Variação de Tokens:** Houve diferença significativa na verbosidade; enquanto o Maritaca gastou ~1.100 tokens, o Claude utilizou entre 3.800 e 4.200 tokens para o mesmo resultado[cite: 11, 15, 16].
* [cite_start]**Escolha de Ferramenta:** O Gemini foi identificado como o melhor para prototipagem rápida, enquanto o Claude é indicado para códigos mais complexos[cite: 17, 18].

## 🔧 Como Executar
1. [cite_start]Copie o **Prompt Estruturado em XML** definido no protocolo de execução[cite: 7].
2. [cite_start]Insira o prompt em um dos modelos de IA testados (ex: Gemini ou Claude)[cite: 6].
3. [cite_start]O modelo irá gerar uma página HTML Single Page com CSS integrado funcional[cite: 5, 7].
