# 🧠 Laboratório de Classificação Visual: Viés e Ética

## 📝 Descrição do Projeto
Este projeto apresenta um experimento prático realizado no **Teachable Machine (Google)** para demonstrar como o viés algorítmico é construído a partir de bases de dados não representativas. O objetivo é treinar um modelo de classificação de perfis profissionais ("Liderança" vs "Operacional") utilizando um dataset deliberadamente enviesado para observar e documentar falhas de inferência e preconceitos automatizados.

Desenvolvido como parte da disciplina de **Inteligência Artificial (2024.1)**, o exercício evidencia que a "inteligência" da máquina é limitada pela qualidade e diversidade dos dados fornecidos, resultando em classificações que replicam estereótipos em vez de competências reais.

## 📊 Memorial de Impacto e Ética
Análise crítica do experimento técnico com foco no impacto humano:

* **Mecanismo do Viés:** A seleção restrita de dados **corrompe** a lógica do algoritmo, pois o modelo **aprende** correlações superficiais (como vestimenta ou gênero) em vez de critérios técnicos. Isso **gera** uma visão distorcida onde a máquina apenas **replica** e **amplifica** preconceitos humanos inseridos no dataset original.
* **Consequência Social:** O sistema **marginaliza** indivíduos qualificados que não **correspondem** ao padrão estético treinado. Essa invisibilização **provoca** exclusão em processos seletivos e **gera** um sentimento de desvalorização profissional e injustiça no indivíduo rotulado incorretamente.
* **Ação Mitigadora:** A intervenção *Human-in-the-loop* **garante** a equidade através de uma curadoria diversa. Uma equipe multidisciplinar **revisa** as categorias antes do treino e **audita** os resultados periodicamente, **assegurando** que o julgamento humano ético **sobreponha** as falhas técnicas de generalização.
