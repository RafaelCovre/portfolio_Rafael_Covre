# 🎮 Batalha de Modelos & Engenharia de Prompt (XML)

## 📝 Descrição do Projeto
[cite_start]Este projeto consiste em um estudo comparativo de **Engenharia de Prompt** utilizando uma estrutura baseada em **XML** para a criação de uma interface web[cite: 2, 5]. [cite_start]O objetivo principal foi avaliar o desempenho de diferentes modelos de IA generativa na geração de uma página HTML Single Page para uma **Loja de Jogos de Videogame e Computador**[cite: 7].

[cite_start]Desenvolvido como parte da disciplina de **Engenharia de Prompt e Aplicações em IA**, o trabalho utilizou um protocolo de execução estruturado para testar diretrizes de design, como layout responsivo e paleta de cores (preta, azul e branca), em diversas ferramentas do mercado[cite: 1, 7]. [cite_start]O sistema buscou identificar qual IA oferece a melhor precisão funcional e criatividade sob restrições técnicas específicas[cite: 9, 11].

## 🚀 Tecnologias Utilizadas
* [cite_start]**Técnica de Prompting:** XML Structured Prompting[cite: 5, 7].
* [cite_start]**Linguagens Geradas:** HTML5 e CSS3[cite: 7].
* [cite_start]**Modelos de IA Testados:** ChatGPT, Gemini, Claude, Qwen, DeepSeek, Grok e Maritaca[cite: 6].
* [cite_start]**Ferramentas de Análise:** Quadro de Análise Comparativa de Precisão e Tokens[cite: 8, 9, 11].

## 📊 Resultados e Aprendizados
[cite_start]O projeto alcançou resultados detalhados sobre a eficiência de cada modelo, demonstrando que a estrutura XML impacta diretamente a compreensão da tarefa[cite: 13].

* [cite_start]**Compreensão Superior:** Os modelos **Gemini e Grok** demonstraram a maior compreensão da estrutura do prompt em XML[cite: 14].
* [cite_start]**Variação de Verbosidade:** Houve diferença significativa na quantidade de tokens gerados para o mesmo objetivo[cite: 15, 16]. [cite_start]O modelo **Claude** foi o mais verboso (até 4.200 tokens), enquanto o **Maritaca** foi o mais econômico (~1.100 tokens)[cite: 11].
* [cite_start]**Casos de Uso:** O **Gemini** foi identificado como a melhor ferramenta para **prototipagem rápida**, enquanto o **Claude** destacou-se para o desenvolvimento de **códigos mais complexos**[cite: 18].

## 🔧 Como Executar
1. [cite_start]Utilize o **Prompt Estruturado** fornecido no protocolo[cite: 7].
2. [cite_start]Insira o código XML em uma das ferramentas de IA (ex: Gemini ou Claude)[cite: 6].
3. [cite_start]O modelo retornará o código HTML/CSS completo para a loja de jogos[cite: 7].
